# Jonas' dotfiles

## Getting started
- `git clone https://github.com/jonaskhalil/dotfiles.git ~/.dotfiles`
- `cd ~/.dotfiles`
- `./install`

## Dotbot
dotbot/ .............. lightweight dotfiles installer
